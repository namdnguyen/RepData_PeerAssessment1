What is mean total number of steps taken per day?

daily <- df %>%
  na.omit() %>%
  group_by(date) %>%
  summarize(total = sum(steps))

ggplot(data=daily, aes(daily$total)) +
  geom_histogram(binwidth = 1000, fill="steelblue") +
  theme_minimal() +
  labs(title = "Histogram of Total Daily Steps", x = "Total Daily Steps", y = "Count")

ave.steps <- format(mean(daily$total), scientific = FALSE)
median.steps <- format(median(daily$total), scientific = FALSE)

The mean total number of steps taken each day is 10766.19 and the median total number of steps taken per day is 10765.

What is the average daily activity pattern?

interval.ave <- df %>%
  na.omit() %>%
  group_by(interval) %>%
  summarize(ave = mean(steps))

ggplot(data = interval.ave, aes(x = interval, y = ave))+
  geom_line(color = "#00AFBB", size = 2) +
